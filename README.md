# FastAPI Auth Backend (Production Ready)

A production-ready FastAPI backend with secure JWT authentication,
refresh tokens, and PostgreSQL support.
Designed for MVPs and early-stage startups.

## Features
- JWT Authentication (Access + Refresh tokens)
- Rotating refresh tokens
- Secure password hashing
- User registration & login
- Protected routes
- PostgreSQL integration
- FastAPI Swagger docs

## Tech Stack
FastAPI • PostgreSQL • SQLAlchemy • JWT • Python

## Core Endpoints
POST /auth/register
POST /auth/login
POST /auth/refresh
GET  /users/me
GET  /health
